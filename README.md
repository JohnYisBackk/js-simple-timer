# Simple Timer PRO

A clean and minimal timer built with **HTML, CSS and vanilla JavaScript**.

This project demonstrates how to build a simple timer using JavaScript, including start, pause, reset functionality and a dark mode toggle.

---

## 🚀 Live Demo

🔗 https://YOUR_USERNAME.github.io/simple-timer-pro/

---

## ✨ Features

* ⏱️ Start timer
* ⏸️ Pause timer
* 🔄 Reset timer
* 🌙 Dark mode toggle
* ⏰ Timer displayed in **HH:MM:SS** format
* ⚡ Accurate timing using `Date.now()`

---

## 🛠️ Built With

* HTML5
* CSS3
* Vanilla JavaScript
